# 브랜드 로고 자산

앱(온보딩 헤더·사이드바)이 **실제 solideo 로고 이미지**를 이 폴더에서 불러옵니다.

아래 파일명으로 저장해 주세요(투명 배경 PNG 권장):

| 파일명 | 용도 | 배경 |
|---|---|---|
| `solideo_logo.png` | 온보딩 헤더(흰 카드 위) | 어두운 글자 버전(첨부 원본) |
| `solideo_logo_white.png` | 사이드바(마룬 배경 위) | 흰색 버전(없으면 기본본 사용) |

파일이 없으면 텍스트 락업(`solideoS.`)으로 자동 폴백합니다.
권장 높이: 헤더 26~30px, 사이드바 22~26px 기준으로 자동 스케일됩니다.
